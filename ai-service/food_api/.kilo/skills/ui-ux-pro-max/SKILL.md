# ui-ux-pro-max

Comprehensive design guide for web and mobile applications. Contains 67 styles, 161 color palettes, 57 font pairings, 99 UX guidelines, and 25 chart types across 16 technology stacks. Searchable database with priority-based recommendations.

## Usage

### Generate Design System (start here)

```bash
python .kilo/skills/ui-ux-pro-max/scripts/search.py "<product keywords>" --design-system -p "Project Name"
```

### Domain search

```bash
python .kilo/skills/ui-ux-pro-max/scripts/search.py "<keyword>" --domain style
python .kilo/skills/ui-ux-pro-max/scripts/search.py "<keyword>" --domain color
python .kilo/skills/ui-ux-pro-max/scripts/search.py "<keyword>" --domain typography
python .kilo/skills/ui-ux-pro-max/scripts/search.py "<keyword>" --domain ux
```

### Stack guidelines

```bash
python .kilo/skills/ui-ux-pro-max/scripts/search.py "<keyword>" --stack html-tailwind
python .kilo/skills/ui-ux-pro-max/scripts/search.py "<keyword>" --stack react
```
